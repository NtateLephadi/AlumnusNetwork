## Overview
This mobile-friendly web app serves as a hub for UCT SCF alumni, allowing them to **post notifications**, **make donations**, and **advertise events**.

## Features
- **Notifications:** Users can share announcements and updates with the alumni community.
- **Donations:** Provides direct banking details for simple contributions.
- **Event Advertising:** Promote events with RSVP tracking, financial contributions, and event details.

## User Flow
1. **Login/Sign-up:** Users authenticate via email or social accounts.
2. **Dashboard:** A central feed displaying notifications and upcoming events.
3. **Create a Post:** Users can submit notifications or event advertisements.
4. **Make a Donation:** Users access the banking details section for direct contributions.
5. **Event Management:** Users can view and RSVP to events, track donations, and explore event details.
